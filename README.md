🌍 GlobeQuest

Discover countries, explore travel possibilities, and plan your next adventure.

✨ Project Description

GlobeQuest is a travel exploration web application that allows users to search and filter countries, view important details about them, estimate travel costs, and build a wishlist of dream destinations.
The goal of this project is to create an interactive platform that makes exploring the world easier and more engaging.

🚀 Features

🔎 Search countries by name
🌍 Filter countries by region
👥 Filter countries by population size
🏳️ View country information (flag, capital, language, population)
💸 Estimated travel cost for destinations
✈️ Flight option integration
❤️ Save destinations to wishlist

🛠️ Tech Stack

Frontend:
HTML
CSS
JavaScript

API Used:
REST Countries API
Amadeus Flight Offers API


⚙️ How the App Works:
The app fetches country data from an API.
Countries are displayed as interactive cards.
Users can search or filter countries.
Travel cost is estimated based on predefined logic.
Flight options are retrieved using an API.
Users can save countries to their wishlist.
