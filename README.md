# 🎬 GlobeQuest – Movie Explorer

GlobeQuest is a simple movie search web application that allows users to search for movies and view basic details like the movie poster, title, and year.
It uses an external movie API to fetch real-time movie data.

---

## 📌 Features

* 🔎 Search for movies by title
* 🎬 Display movie posters and details
* 📅 Filter movies by year range
* 🔤 Sort movies by title or year
* 📱 Responsive movie grid layout
* 🎨 Clean dark-themed UI

---

## 🛠️ Technologies Used

* **HTML5** – Website structure
* **CSS3** – Styling and layout (Flexbox + Grid)
* **JavaScript (Vanilla JS)** – Fetching and displaying movie data
* **Movie API** – For retrieving movie information

## 🚀 How to Run the Project

1. Download or clone the repository.

```
git clone https://github.com/your-username/globequest-movie-explorer.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

No installation or server is required.

---

## 🔍 How It Works

1. The user enters a movie name in the search bar.
2. JavaScript sends a request to the movie API.
3. The API returns movie data.
4. The movies are displayed in a responsive grid layout.
